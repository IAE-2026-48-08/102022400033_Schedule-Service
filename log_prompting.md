# Rekap Log Prompting AI

## Daftar Prompt yang Digunakan

1. Bagaimana cara membuat Schedule Service menggunakan Laravel dan SQLite?

2. Bagaimana cara mengimplementasikan REST API dengan endpoint:

   * GET /api/v1/schedules
   * GET /api/v1/schedules/{id}
   * POST /api/v1/schedules

3. Bagaimana cara menerapkan Standard Integration Contract pada response JSON?

4. Bagaimana cara mengimplementasikan API Key menggunakan header X-IAE-KEY pada Laravel?

5. Bagaimana cara membuat middleware untuk melakukan autentikasi API Key?

6. Bagaimana cara mengintegrasikan Swagger/OpenAPI menggunakan package L5-Swagger?

7. Bagaimana cara mendokumentasikan endpoint REST API menggunakan OpenAPI Attributes?

8. Bagaimana cara menghasilkan dokumentasi interaktif Swagger yang menampilkan seluruh endpoint Schedule Service?

9. Bagaimana cara mengintegrasikan GraphQL menggunakan package Lighthouse?

10. Bagaimana cara membuat schema GraphQL untuk entitas Schedule?

11. Bagaimana cara membuat Query GraphQL untuk mengambil seluruh data Schedule?

12. Bagaimana cara membuat Query GraphQL untuk mengambil data Schedule berdasarkan ID?

13. Bagaimana cara mengintegrasikan GraphQL Playground untuk melakukan pengujian query?

14. Bagaimana cara melakukan pengujian REST API dan GraphQL untuk memastikan seluruh layanan berjalan dengan baik?
